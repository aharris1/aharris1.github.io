Project Successes

  • Familiarized myself with RESTful API architecture in only 1 week.
  • Conceived and created application for contest entry in only one week, while still attending undergraduate classes.
  • Got application working despite server-side errors derailing testing procedures at the last minute.
  • Received further commendation from contest organizers in the form of continuing codebase contributions after the results were announced.

Project Failures

  • Did not update the project after the game developed changed the API, rendering the current version nonfunctional.

Lessons Learned and Skills Developed

  • Java Coding - While I had experience with programming in Java through my AP Computer Science course in high school, I had not previously attempted to develop an application of this size to be released to other users before. In the process I learned about methods for making sure that Java programs could be run on other users' systems, including avoiding any use of absolute path references in favor of relative paths.
  • RESTful API Architectures - RESTful application programming interfaces (API) make up the backbone of the internet through the hyper-text transmission protocol, providing a flexible method for programmatically requesting data from servers for integration into a client-side application. I had no prior experience working with such APIs, and in the course of this project I rapidly acquainted myself with them and developed hands-on experience with making valid requests.
  • Single Sign-On Authentication - Some of the functions I needed to utilize for my application required that the user provide permission, which required that I program a method to handle the single sign-on (SSO) authentication protocol that is commonly used to allow tokens from one sign-on service to be securely used on another. Examples of SSO include the increasingly common "Sign in with Google" option on websites, and so developing hands-on experience with coding the HTTP requests necessary to implement the SSO handshake protocol provides a useful understanding of a common internet authentication protocol.
  • Fast-Paced Coding Practices - The short development period required that I further develop my skills with rapidly iterating through different approaches and following the most promising one. In the process of doing so, I gained experience with distinguishing between a necessary feature and a premature optimization, which enabled me to spend my limited time on the former and avoid squandering it on the latter.
